# ic-presentation-template (UNICAMP)

A modularized LaTeX presentation template designed for the **Institute of Computing (IC)** community. 

This project moves away from dated default aesthetics, providing a clean, flat-design UI that is high-contrast and easy to maintain.

## 🚀 Overleaf Verified (March 31, 2026)
- **Status:** Stable & Error-Free.
- **Compiler:** pdfLaTeX.
- **TeX Live:** 2025/2026 (Standard Overleaf Environment).

## ✨ Features
- **IC-Modular Footer:** A custom-built bar that handles long names and affiliations (e.g., "Luis Vásquez | IC -- UNICAMP") with no layout overlap.
- **Modular Color Palette:** Control your entire theme (Primary, Secondary, Accent) from one configuration block.
- **Scientific Ready:** Pre-configured with `chemfig` and `mhchem` for chemical modeling and formulas.
- **Minimalist Aesthetic:** Strips away the "Beamer-default" look for a modern, professional feel.

## 🛠️ How to Use
1. Start a "Blank Project" in **Overleaf**.
2. Copy the code from `src/template.tex` into your project.
3. Update your name, title, and date in the **USER SETTINGS** section at the top.
4. Hit **Recompile**.
